## 使用说明

## 一、用IDEA打开项目
### 1. clone代码
  git clone https://github.com/Jade-kin/JavaTest2.git
### 2. 用IDEA代开项目，等待依赖导入完毕后打开Terminal
  点击Terminal
### 3. 打包项目生成jar包并运行jar包
  sh build.sh
  
## 二、在文件夹中打开项目
### 1. clone代码
  git clone https://github.com/Jade-kin/JavaTest2.git
### 2. 进入项目文件夹后鼠标右击，打开git Bash Here
  git Bash Here
### 3. 打包项目生成jar包并运行jar包
  sh build.sh
